Demo Mode

It appears that this application is running on a virtual or cloud-based system, as it lacks physical sensors. This application is intended to be ran on a physical server. Some of the data processed by WHM is not applicable or it's not available in the same way as it would be on a physical server. In these cases, sample data is processed in the same way as genuine information. This allows for testing and demonstration of all functionality.

Sample data is used for the following:

  • Sensors
  • Storage
  • Host Logs

Also, the "Restart" button above might not actually restart the system.

Web Host Manager

WHM is a one stop shop for checking up on your server. Forget setting aside time to ssh into your server and google-ing diagnostic commands for the nth time to ensure your system is still healthy. WHM runs and tracks your diagnostics for you so you can check your stats at a glance and get back to what's important.

You are registered as an admin. You are able to view all available information, make change WHM settings, excecute simple tasks on the system, and manage users.

You are registered as a standard user. You will be able to see all diagnostic information, but some other features will not be available. Contact an admin if you need more access.

Current diagnostic information is loaded when you log in. It can be viewed from the tabs on the navigation bar. Click the refresh button on the navigation bar to re-run diagnostics and display updated information.

The logout button is located under your user name on the navigation bar.
